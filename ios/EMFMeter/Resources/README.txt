AUDIO RESOURCES

This folder should contain:

1. geiger_click.mp3 (or .wav, .m4a)
   - A short Geiger counter click sound effect
   - Recommended duration: 50-100ms
   - Should be a single click/tick sound
   - Free alternatives available from:
     - freesound.org (search "geiger click")
     - zapsplat.com
     - pixabay.com

To add the audio file:
1. Download or create a suitable click sound
2. Name it "geiger_click.mp3" (or appropriate extension)
3. Place it in this folder
4. Add it to the Xcode project (drag into Resources group)

ASSET CATALOG

You should also create an Assets.xcassets folder with:
- AppIcon (app icon in various sizes)
- AccentColor
- LaunchScreenBackground (optional)
